# PRD: Галерея-магазин Ирины Кречетовой

## 1. Overview

### Продукт
Сайт-галерея и интернет-магазин художника-импрессиониста Ирины Кречетовой с двумя направлениями: живопись и лэмпворк (авторские стеклянные украшения и декор для дома).

### Big Job
Помочь ценителям искусства найти и приобрести уникальные авторские работы — картины и изделия из стекла — для интерьера, подарков и коллекций, а художнику — выйти на российский, европейский и китайский рынки.

### Целевые рынки
- Россия
- Европа
- Китай

### Языки
- Русский
- English
- 中文 (китайский)

---

## 2. Целевые сегменты

### Сегмент 1: Дизайнеры интерьера
**Core Jobs:**
- **Когда** работаю над дизайн-проектом, **хочу** быстро найти оригинальную картину, подходящую по стилю, цветовой гамме и размеру
- **Когда** клиент хочет «живой» интерьер, **хочу** предложить авторскую работу с возможностью увидеть её в раме
- **Когда** нужно укомплектовать несколько помещений, **хочу** заказать несколько работ у одного художника

**Big Job:** Создавать интерьеры, которые вызывают эмоции, чтобы укреплять репутацию и получать рекомендации

### Сегмент 2: Ценители искусства (Россия + Европа + Китай)
**Core Jobs:**
- **Когда** обустраиваю жильё, **хочу** найти авторскую картину импрессиониста, создающую атмосферу
- **Когда** ищу подарок, **хочу** подарить оригинальное украшение ручной работы
- **Когда** знакомлюсь с культурой другой страны, **хочу** приобрести произведение искусства

**Big Job:** Окружить себя красивыми уникальными вещами, чтобы чувствовать гармонию и выражать свой вкус

### Сегмент 3: Покупатели авторских украшений и декора для дома
**Core Jobs:**
- **Когда** хочу выделиться, **хочу** найти украшение, которого нет ни у кого
- **Когда** ищу подарок, **хочу** подарить красивое изделие ручной работы с историей
- **Когда** украшаю дом к праздникам, **хочу** найти уникальные ёлочные игрушки и интерьерный декор из стекла

**Big Job:** Выражать индивидуальность через уникальные аксессуары и декор

## 3. Рискованные предположения (RAT)

### RAT #1 — Готовность покупать онлайн у неизвестного художника (Score: 20)
- **Предположение:** Конверсия ≥ 2–5% из посещения в заказ при трафике 1 000+ уников/мес
- **Риск:** Инвестиции в мультиязычность и международную логистику не окупятся
- **Категория:** Ценность / готовность купить
- **P:** 4, **I:** 5
- **Валидация через MVP:** Запуск сайта с аналитикой, таргет на EN/CN аудиторию

### RAT #2 — Дизайнеры предпочтут прямой заказ через сайт (Score: 12)
- **Предположение:** ≥ 10 дизайнеров за 6 месяцев сделают повторный заказ, средний чек ≥ €500
- **Риск:** Без личных контактов сегмент не конвертируется
- **Категория:** Сегмент (экономическая привлекательность)
- **P:** 3, **I:** 4
- **Валидация через MVP:** Отдельная посадочная для дизайнеров, партнёрская программа

### RAT #3 — Лэмпворк обеспечит устойчивый поток выручки (Score: 9)
- **Предположение:** ≥ 300 заказов/год, средний чек €30–150, ≥ 30% выручки
- **Риск:** Высокая конкуренция, сезонность ёлочных игрушек
- **Категория:** Юнит-экономика
- **P:** 3, **I:** 3
- **Валидация через MVP:** Отдельный блок лэмпворка на сайте с аналитикой по конверсии

## 4. MVP — Функциональные требования

### 4.1 Каталог картин
- Сетка карточек с фото картин
- **Hover-эффект:** картина без рамы → при наведении курсора — картина в раме
- Под каждой картиной: название, размер (ШxВ см)
- Цена с автоматической конвертацией валюты (₽ / € / ¥) в зависимости от языка
- Статус: «В наличии» / «Продано» (с пометкой)
- Для проданных: кнопка «Заказать копию» → форма запроса (срок изготовления обсуждается индивидуально)
- Фильтры и сортировка по категориям

### 4.2 Каталог лэмпворка
- Аналогичная сетка карточек
- Категории: украшения (для женщин, для мужчин), декор для дома, ёлочные игрушки
- Возможность заказа нескольких экземпляров (для серийных изделий — ёлочные игрушки и т.п.)
- Уникальные изделия — в единственном экземпляре

### 4.3 Корзина и оплата
- Добавление товаров в корзину
- Оформление заказа с указанием адреса доставки
- Универсальная платёжная система (Stripe или аналог с поддержкой международных карт)
- Стоимость доставки: «рассчитывается индивидуально» (менеджер связывается с покупателем)

### 4.4 Мультиязычность
- 3 языка: русский, английский, китайский
- Переключатель языка в шапке сайта
- Автоматическая конвертация цен:
  - Русский → ₽
  - English → €
  - 中文 → ¥

### 4.5 Личный кабинет покупателя
- Регистрация / авторизация
- История заказов
- Статус текущих заказов
- Сохранённые данные для доставки

### 4.6 Админ-панель
Поля для каждого товара:
- Фото картины / изделия (основное)
- Фото в раме (для картин — используется в hover-эффекте)
- Название
- Размер (ШxВ)
- Цена
- Категория (картины / лэмпворк)
- Подкатегория (натюрморт, пейзаж, украшение, ёлочная игрушка и т.д.)
- Статус (в наличии / продано)
- Техника (масло, акрил и т.д.)
- Год создания
- Материалы

### 4.7 Страницы
- Главная (витрина с избранными работами)
- Каталог картин
- Каталог лэмпворка
- Карточка товара (детальная страница)
- Корзина
- Оформление заказа
- Личный кабинет
- О художнике
- Контакты
- Доставка и оплата

## 6. Дизайн

### Цветовая гамма
- **Основные цвета:** пастельная бирюза, золото, белый
- **Дополнительные:** гармоничные пастельные оттенки (кремовый, нежно-розовый, светло-серый)
- Чистый, воздушный дизайн, подчёркивающий произведения искусства

### Референс
- [bmwpainting.ru](https://bmwpainting.ru/glavnaya/folder/still_lifes) — структура каталога, карточки товаров, hover-эффекты

### UX-принципы
- Картины и изделия — главный визуальный элемент, интерфейс не конкурирует с ними
- Минималистичная навигация
- Быстрый путь до покупки (≤ 3 клика от каталога до оформления заказа)

## 7. Edge Cases

| Сценарий | Поведение |
|---|---|
| Картина продана | Отображается с пометкой «Продано» + кнопка «Заказать копию» |
| Заказ копии | Форма запроса, срок обсуждается индивидуально |
| Уникальное изделие лэмпворк | Количество = 1, нельзя добавить больше |
| Серийное изделие (ёлочные игрушки) | Можно выбрать количество |
| Доставка | Стоимость рассчитывается индивидуально после оформления |
| Валюта | Автоматическая конвертация по выбранному языку |

## 8. Метрики успеха MVP

| Метрика | Целевое значение |

| Конверсия посещение → заказ | ≥ 2% |
| Повторные заказы от дизайнеров (6 мес) | ≥ 10 |
| Доля выручки от лэмпворка | ≥ 30% |
| Заказы лэмпворк / год | ≥ 300 |
| Средний чек (картины) | ≥ €500 |
| Средний чек (лэмпворк) | ≥ €50 |
